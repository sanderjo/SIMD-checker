# SIMD-checker: does a Linux binary contain SIMD code?

Works on x86, ARM and RISC-V.

```
$ ./SIMD_checker.sh ~/nzbget/nzbget
ELF binary. Good.
Detected x86_64
Match found: SIMD
```

```
./SIMD_checker.sh /usr/lib/python3/dist-packages/sabctools/sabctools.cpython-313-x86_64-linux-gnu.so
ELF binary. Good.
Detected x86_64
Match found: SIMD
```

```
./SIMD_checker.sh /usr/bin/python3.12
ELF binary. Good.
Detected ARM64 (AArch64)
Match found: SIMD
```


```
./SIMD_checker.sh /usr/bin/python3.12
ELF binary. Good.
Detected RISC-V
Match found: SIMD

```

# On ARM64 aka Aarch64
```
gcc -O3 -march=native -ftree-vectorize -o simd_example simd_example.c
```

